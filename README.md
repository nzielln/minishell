# Minishell 
A mini shell written using C. 

## Commands
### Built-in Commands (in addition to standard commands)
- `cd`
- `help`
- `exit`
- `history`
- `game`
- `tab`

## Build Commands
- Navigate to the the project root directory
- Run command `mkdir build && cd build`
- Build shell using CMake, `cmake ..`, then `make`
- Run shell: `./minishell`
